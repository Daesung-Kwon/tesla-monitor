## ✅ 체크리스트

### 1단계: Telegram Bot 생성 (3분)
```
[ ] @BotFather에게 /newbot
[ ] Token 받기: 123456789:ABC...
[ ] @userinfobot에게 /start
[ ] Chat ID 받기: 987654321
```

### 2단계: GitHub 설정 (3분)
```
[ ] GitHub에 Repository 생성
[ ] 로컬 코드 푸시: git push
[ ] Secrets 추가:
    - TELEGRAM_BOT_TOKEN
    - TELEGRAM_CHAT_ID
```

### 3단계: GitHub Actions 활성화 (1분)
```
[ ] Actions 탭 → Enable workflows
[ ] Run workflow (수동 실행)
[ ] 로그 확인: "첫 실행 - 초기 데이터 저장"
```

### 4단계: 15분 후 확인
```
[ ] 자동 실행 확인 (Actions 탭)
[ ] Telegram에서 메시지 대기
[ ] 완료! 🎉
```
