# 🕐 시간대(Timezone) 설명

## 📊 현재 상태

### RSS 피드별 시간대

각 뉴스 소스는 **자신들의 현지 시간**으로 기사 시간을 제공합니다:

| 소스 | 본사 위치 | 시간대 | 예시 |
|------|----------|--------|------|
| **Tesla Blog** | 미국 캘리포니아 | PST/PDT (UTC-8/-7) | `2026-01-18 10:00 PST` |
| **Electrek** | 미국 | EST/EDT (UTC-5/-4) | `2026-01-18 13:00 EST` |
| **Teslarati** | 미국 | PST/PDT (UTC-8/-7) | `2026-01-18 10:00 PST` |
| **InsideEVs** | 미국 | EST/EDT (UTC-5/-4) | `2026-01-18 13:00 EST` |
| **Tesla North** | 캐나다 | EST/EDT (UTC-5/-4) | `2026-01-18 13:00 EST` |

### 현재 Telegram 알림

```
📅 2026-01-18 10:00  ← 이게 PST 시간인지, EST 시간인지 모호
```

**문제점:**
- ❌ 시간대 정보 없음
- ❌ 뉴스마다 다른 시간대 혼재
- ❌ 한국 시간(KST)과 최대 17시간 차이

**한국 시간과 차이:**
- PST: 한국보다 **17시간 느림** (겨울)
- PDT: 한국보다 **16시간 느림** (여름)
- EST: 한국보다 **14시간 느림** (겨울)
- EDT: 한국보다 **13시간 느림** (여름)

### 예시: 혼란스러운 상황

```
알림 #1: Tesla Blog
📅 2026-01-18 10:00  ← PST (실제 한국 시간: 01/19 03:00)

알림 #2: Electrek
📅 2026-01-18 13:00  ← EST (실제 한국 시간: 01/19 03:00)

→ 같은 시간에 발행되었는데 표시는 다름!
```

## ✅ 해결 방법: 한국 시간(KST) 통일

모든 기사 시간을 **한국 시간(KST, UTC+9)**으로 변환해서 표시!

### Before (현재)

```
...
```

### After (개선)
```
🚗⚡ 테슬라 뉴스 업데이트!

Tesla Cybertruck production doubles

📰 출처: Electrek
📅 2026-01-19 03:00 KST 🇰🇷  ← 한국 시간 명확!

...
```

## 🔧 기술 구현

### 변환 로직

```python
from datetime import datetime
import pytz

# 1. RSS 시간 파싱 (시간대 포함)
published = "Sat, 18 Jan 2026 10:00:00 PST"
pub_date = datetime.strptime(published, '%a, %d %b %Y %H:%M:%S %Z')

# 2. 한국 시간(KST)으로 변환
kst = pytz.timezone('Asia/Seoul')
kst_time = pub_date.astimezone(kst)

# 3. 포맷팅
date_str = kst_time.strftime('%Y-%m-%d %H:%M KST 🇰🇷')
# → "2026-01-19 03:00 KST 🇰🇷"
```

### 장점
- ✅ 모든 기사를 한국 시간으로 통일
- ✅ 시간대 혼란 제거
- ✅ 직관적 (한국에서 볼 때)
- ✅ "어제 밤에 나온 기사구나!" 바로 이해

## 📊 실제 예시

### 시나리오: 같은 시간에 발행된 기사 3개

**원본 (RSS 피드):**
```
Tesla Blog:    2026-01-18 10:00:00 PST
Electrek:      2026-01-18 13:00:00 EST  
Teslarati:     2026-01-18 18:00:00 UTC
```

→ 전부 **같은 순간**이지만 표시가 다름!

**현재 (시간대 없이):**
```
Telegram 알림:
📅 2026-01-18 10:00  ← Tesla Blog
📅 2026-01-18 13:00  ← Electrek
📅 2026-01-18 18:00  ← Teslarati
```
❌ 3시간, 8시간 차이나는 것처럼 보임!

**개선 (KST 통일):**
```
Telegram 알림:
📅 2026-01-19 03:00 KST 🇰🇷  ← Tesla Blog
📅 2026-01-19 03:00 KST 🇰🇷  ← Electrek
📅 2026-01-19 03:00 KST 🇰🇷  ← Teslarati
```
✅ 같은 시간임을 명확히 알 수 있음!

## 🌍 다른 옵션들

### Option 1: 한국 시간 (KST) - 추천! ✅
```

```
- 한국 거주자에게 직관적
- 모든 기사 시간 통일

### Option 2: 원본 시간 + 시간대 표시
```
📅 2026-01-18 10:00 PST (한국: 01/19 03:00)
```
- 원본 정보 유지
- 하지만 메시지가 길어짐

### Option 3: UTC 통일
```
📅 2026-01-18 18:00 UTC
```
- 국제 표준
- 하지만 한국인에게 직관적이지 않음

### Option 4: 상대 시간
```
📅 2시간 전
```
- 직관적
- 하지만 정확한 시간 모름
